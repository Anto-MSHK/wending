# Story 1.4: Core Content Sections

Status: ready-for-dev

## Story

As a Guest,
I want to see the event details, map, and countdown,
so that I get excited and know the logistics.

## Acceptance Criteria

1. **Given** The user is on the landing page
2. **When** They scroll down
3. **Then** They see the "Event Details" card (Date: June 15, 2026, Time: 14:00, Venue: The Grand Estate)
4. **And** They can tap a "View Map" link (Opening Google Maps)
5. **And** They see a live Countdown Timer ticking down to the event date

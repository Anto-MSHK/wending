Status: complete

## Story

As a Guest,
I want to see my name immediately upon loading the page,
so that I know this invitation is specifically for me.

## Acceptance Criteria

1. **Given** A valid guest token (passed from Middleware/URL)
2. **When** The Landing Page loads (Server Component)
3. **Then** The Guest's name is displayed prominently in the Hero section
4. **And** The "Love Story" introductory text is visible
5. **And** The page uses the defined typography and color palette (Blush/Gold/Cream)
